# Jira Sync

## Purpose
Synchronize project state and tasks with Jira.

## Workflow
1. Connect to Jira via MCP or API.
2. Pull latest ticket statuses or push local updates.
3. Map local epics and stories to Jira issues.
4. Ensure consistency between the codebase docs and Jira.

## Expected Output
Synchronized task status between local documentation and Jira.
