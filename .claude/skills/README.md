# Claude Skills Overview

This directory contains a comprehensive set of specialized skills organized by category. These skills empower Claude to act as a senior expert across various domains of software development.

## Categories

1. **[01-Languages](./01-languages/)**: Core programming language expertise (TypeScript, Python, Go, Swift).
2. **[02-Backend Frameworks](./02-backend-frameworks/)**: Backend development with Node.js, NestJS, FastAPI, Django, and Spring Boot.
3. **[03-Frontend Frameworks](./03-frontend-frameworks/)**: Modern web development with React, Vue, Angular, Next.js, and Astro.
4. **[04-Mobile](./04-mobile/)**: Mobile application development for iOS (SwiftUI, UIKit) and Android (Jetpack, XML), plus React Native.
5. **[05-Database](./05-database/)**: Data persistence and optimization with PostgreSQL, MongoDB, Redis, and Prisma.
6. **[06-API](./06-api/)**: API design and communication (REST, GraphQL, gRPC, WebSockets).
7. **[07-Testing](./07-testing/)**: Quality assurance patterns (Unit, Integration, E2E, Performance, TDD).
8. **[08-DevOps](./08-devops/)**: Infrastructure and automation (Docker, Kubernetes, CI/CD, Terraform, Monitoring).
9. **[09-Security](./09-security/)**: Secure coding and compliance (AuthN, AuthZ, Secure Coding, Pen Testing, Compliance).
10. **[10-Architecture](./10-architecture/)**: System design and patterns (Clean Architecture, DDD, Microservices, Event-Driven, Design Patterns).
11. **[11-Data & ML](./11-data-ml/)**: Data engineering and AI integration (Data Engineering, Machine Learning, AI Integration).
12. **[12-Platform Specialists](./12-platform-specialists/)**: Platform-specific expertise (AWS, Azure, GCP, Vercel, Netlify).

## Usage
Invoke a skill by its name or by referencing its path in your reasoning chain. Skills should be loaded when specific expertise is required for a task.
