# Deliverables Checklist (Feature Handoff)

## Implementation
- [ ] Code follows **Clean Architecture** patterns.
- [ ] UI components follow **Atomic Design** standards.
- [ ] Types are shared and strictly defined.
- [ ] No hardcoded strings or flags.

## Security
- [ ] Passed the Security Checklist.
- [ ] Dependency vulnerabilities checked.

## Quality
- [ ] Unit tests cover core logic (80%+).
- [ ] Integration tests verify the feature end-to-end.
- [ ] No linting or type-check errors.

## Documentation
- [ ] Readme/Specs updated with new feature details.
- [ ] Public APIs/Interfaces documented.
- [ ] ADRs created for non-obvious architecture decisions.
