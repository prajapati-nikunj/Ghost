# TypeScript Pro

Senior TypeScript specialist with deep expertise in advanced type systems and full-stack type safety.

## When to Use This Skill
- Building type-safe full-stack Node.js/TypeScript applications.
- Implementing Clean Architecture with **strict layer boundaries**.
- Domain Modeling with **Branded Types** and **Value Objects**.
- End-to-end type safety with tRPC or Zod validation.

## Core Workflow
1. **Domain (Core)**: Define pure entities and domain errors.
2. **Application (Use Case)**: Implement interactors and service protocols.
3. **Interface (Adapters)**: Create controllers, middlewares, and repositories.
4. **Scaffolding**: Configure `tsconfig` path aliases (e.g., `@core/*`, `@use-cases/*`).
