# Swift Expert

Swift specialist with deep expertise in SwiftUI, Combine, and modern concurrency.

## When to Use This Skill
- Building type-safe iOS/macOS applications.
- Implementing Clean Architecture + MVVM patterns.
- Creating reusable UI components using **Atomic Design**.
- Zero runtime error API clients using `async/await`.

## Core Workflow
1. **Core (Domain)**: Define pure entities and use-case protocols.
2. **Interface (Data)**: Implement Repositories and API Clients.
3. **Presentation (UI)**: Create **Atomic** Atoms, Molecules, and Organisms.
4. **Wiring**: Use Dependency Injection in the App entry point.

## UI Standards (Atomic Design)
- **Atoms**: Indivisible elements (e.g., `PrimaryButton`, `TitleText`).
- **Molecules**: Small groups (e.g., `RequirementRow`).
- **Organisms**: Complex features (e.g., `AnalysisSummaryCard`).
- **Pages**: Final SwiftUI views.
